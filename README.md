# Ozon Clone

A clone of the popular Russian marketplace Ozon, created for learning purpose only.

## Project Description

This project is a web application that mimics the functionality and design of the Ozon online store. The project includes a main page, product catalog, and separate brand pages such as Redmi.

## Project Structure

```
ozon-clone/
├── assets/                 # Main resources and styles
│   └── main.css           # Main styles
├── files/                  # Images and JavaScript files
├── fonts/                  # Fonts
├── icons/                  # Icons
├── styles/                 # Additional styles
│   ├── main.css           # Main site styles
│   └── typography-desktop-6.1.6.css  # Typography styles
├── xml/                    # XML files
├── index.html              # Main page
├── redmi.html              # Redmi brand page
├── desktop.webmanifest     # PWA manifest
└── README.md               # This file
```

## Features

- Main page with product catalog
- Brand pages with products
- Shopping cart
- Product search
- Responsive design for various devices

## Usage

1. Clone the repository to your local computer:
   ```
   git clone https://github.com/yourusername/ozon-clone.git
   ```

2. Navigate to the project directory:
   ```
   cd ozon-clone
   ```

3. Open `index.html` in your web browser:
   ```
   Simply double-click the index.html file or open it through your browser
   ```
